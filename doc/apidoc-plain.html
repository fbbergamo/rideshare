Drivers

Drivers / index

GET /api/v1/drivers
Busca motoristas por lat e lng

Examples

GET /api/v1/drivers?sw=-90%2C-180&ne=90%2C180
200
[
  {
    "DriverId": "419",
    "latitude": -90,
    "longitude": -180,
    "driverAvailable": true
  },
  {
    "DriverId": "420",
    "latitude": -90,
    "longitude": -180,
    "driverAvailable": true
  }
]

Params

  • sw : optional [

    Must be String

    ]

    latitude e longitude separado por virgula ex: -90,-180

  • ne : optional [

    Must be String

    ]

    latitude e longitude separado por virgula ex: -90,-180

Drivers / show

GET /api/v1/drivers/:id
Retorna informacoes relativa ao motorista

Examples

GET /api/v1/drivers/415?%7B%7D
200
{
  "id": 415,
  "plate": "MyString",
  "latitude": null,
  "longitude": null,
  "available": true,
  "name": "MyString"
}

Params

  • id : optional [

    Must be a number.

    ]

    Id do motorista
