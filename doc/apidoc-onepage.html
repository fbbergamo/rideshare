GET /api/v1/drivers
Busca motoristas por lat e lng

Examples

GET /api/v1/drivers?sw=-90%2C-180&ne=90%2C180
200
[
  {
    "DriverId": "419",
    "latitude": -90,
    "longitude": -180,
    "driverAvailable": true
  },
  {
    "DriverId": "420",
    "latitude": -90,
    "longitude": -180,
    "driverAvailable": true
  }
]

Params

Param name Description
sw
optional

latitude e longitude separado por virgula ex: -90,-180


Value:

Must be String
ne
optional

latitude e longitude separado por virgula ex: -90,-180


Value:

Must be String
